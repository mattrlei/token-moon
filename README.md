# Token Moon

AI cost intelligence platform. Pre-build cost estimation for n8n workflows, direct API usage, Claude Code agentic sessions, and Claude.ai subscription planning.

**Live demo:** [tm.mattrlei.com](https://tm.mattrlei.com)

## What it does

Token Moon parses n8n workflow JSON before the workflow runs and projects monthly LLM cost across multi-node architectures. It is the only tool purpose-built for n8n workflow cost estimation.

Four modes:
- **n8n Workflows** — paste workflow JSON, get pre-build cost estimate with loop multiplier detection
- **API Calls** — direct API token cost modeling across Anthropic and OpenAI
- **Claude Code** — agentic session cost projection
- **Plan Advisor** — Claude.ai subscription tier recommendation

## Tech

Static HTML/CSS/JS. No build step. No dependencies. Deployed via GitHub Pages.
